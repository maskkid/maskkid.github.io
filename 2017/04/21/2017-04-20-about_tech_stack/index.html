聊聊技术栈，吹吹水

Posted on 2017-04-21 Edited on 2021-08-13 In tech

why say?

不知道从什么时候开始，不断的学习，不断的玩不同的技术；也不知道从什么时候开始，学习的速度越来越快。自己玩，基本都能实现。那都会些什么呢？简单列一下，回头可能会忘掉，要多回头看看自己走过的路。

概况

目前开发主要使用linux系统+window7系统，软件主要是vim，jetbrains系列，vscode，sublime，其他工具软件很多，3D的，游戏的，2D游戏的。

技术栈列表

技术名称 程度 备注
前端开发 熟悉 js底层知识；各种框架，比较熟悉的jq，zepto，understore，angular1.x，用过的，react，vue，backbone，prototype，sencha，YUI
php 了解，常用 写过原生，用过框架，写过自用框架
服务器 了解 linux常用知识
游戏开发 了解 主要h5游戏，引擎主要是phaser,collie,cocos2d-js,egret.
golang 了解 目前个人最喜欢的语言，一直在做项目。完成的有一套TCP信息采集处理小系统，客户端为web形式，打包单文件运行
python 了解 几年前主力语言之一，自己玩的业余项目基本基于此
java 了解 系统学过，为了android开发，基本能解决使用问题，依赖google
c/cpp 了解 系统学过几本书，做过一些小的利用工具，主要在linux下
c# 了解 为了搞unity3D，系统看过基础，玩过大概半年时间。后期，使用c#做过一个基于gecko（webkit也做了，但是代理切换接口有问题）的浏览器，用于自控制代理和网页注入监测。
其他（树莓派、视频流媒体服务器、ffmepg、opencv等） 玩过 爱好很多，有一部分为了给点点同学做一些教程玩具
