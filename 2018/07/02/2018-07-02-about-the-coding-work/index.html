许久不见

之前一直在忙，切到mac后，一直没有搞jekyll的环境。博客也就懒惰的暂停了，期间玩兴小发，做了一个自己的个人php+纯语义html版版轻博客,写了一段时间骚气的文学小品。

既然好久不见，补补这一年的学习经历。

在开发上，主要放在前端和PHP开发两方面。

  1. 工作以前端为主，前半年用了vue，期间对项目结构做了整体的定制, 基本围绕工程化、基础库、业务层规划、定制化业务开发几方面来做，其他语言和框架方案也差不多；后面用了一段时间12年的老架构方案，jQuery+自扩展全家桶；目前换react+redux+typescript这一套。这方面，主要对typescript和react加深了应用经验。之前由于对java/c#和其他语言都了解，对typescript不是很看好，认为是对前端发展的一种阻碍。没想到，前端这群人太能闹腾，跟风变化的非常快，没办法，应景的用了一阵子，算是踩着时代的尾巴吧。

  2. PHP开发，前后都做，以会议系统、发布系统为主，根据项目大小不同，不同的框架方案，做了有十来套。这里主要应用了一下CI/CD和docker，目前个人的项目除了coding之外，其他都交给了自动化处理。

玩的方面，主要以golang为主，业余时间不多，断断续续做了几个工具，最近的项目是前端react+redux+typescript这套东西出了一套代码生成器。

over，总的来说还好，产出不敢说,but。。。。平均睡眠时间比上年少了很多 :(

吐槽下前端各种框架和工具，呼。。。不说了
