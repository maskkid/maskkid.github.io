8年来首次参加系列面试-营养

Posted on 2016-03-04 Edited on 2021-08-13 In other 
    条条大路通罗马异或殊途同归，选择不一定找到最好，但是会找到当下最适合的。得失参半，半参半悟。

    从年前到年后，经过不到10家公司面试，有的经过3-4面，但仅记录下遇到的比较深刻的人和收获。相比面对电脑，需要的更多是学而悟和交流。

他们

  • boy，天津圈子有幸结识的朋友。经历丰富，技术精湛，朋友多，人谦逊。很多的机会都是他的二度人脉。
  • sy，邓哥。出身顶尖学府，系列顶尖公司经历，曾经的底层语言开发者，创造给前端使用的浏览器开发。为人谦和，像遇到的学识丰富的前辈一样，让人佩服。如果不是解决生存的待遇问题，一定会跟他干几年。
  • sy，x姐。出身顶尖学府，健谈，谦和略傲娇。算法大拿，经验丰富，待人老练。
  • hb，胖胖前端哥。人很好，像技术男，大家聊天没有障碍，言语不多，实实在在。如果去，也是因为他，一定会多个朋友。
  • hb，阿里男。技术应该挺牛，聊了几句，感觉“异常”傲娇，应该是技术优而管，管而傲。如果不去，因为他。每个人会有感性和理性的验证标准，我的判断可能有误，但步止于此。
  • wy，白面书生。因为朋友没在，临时换成了这个人，应该是头目，他没介绍，我也没问。1分钟看完简历，不甩。现在回想，当时已经不行了。但是又问了很多基础技术和知识面的问题。还有几个理解前端开发的问题。为来这个公司，跑了一个半小时，面试从头到尾10分钟。
  • qy，几面遇到的人都不错，公司感觉也很好，第一个参加面试的公司。标准的流程，登记，前台给笔试题，答完，一面，过关，二面，过关，三面待遇，四面hr待遇细则。但是没过，过了应该也不会去，待遇达不到。
  • ymjr，技术男。硕士，年龄不大，比较沉稳，有工作激情，聊的很好。但是在加班的问题上，没共识。如果7年前，我们也是一类人。但是小公司给我的刺激较大，加班累成狗，病了自己看，请假还扣钱，用命在换钱。
  • my，张总。人情把握好，有从业战略眼光，健谈，跟曹勇军有部分相似。
  • tf，旋律。技术男，跟boy挺像，没准会共事，再处处。
  • tf，赵总。技术转型，健谈，有领导力，有魄力，有想法，年轻有为。
  • yb，穆总。从业丰富，老练，很有领导力，办事雷厉风行。应该是摸爬滚打多年，有几个得力干将，跟随多年。是个我眼中做大事儿的。
  • ls，一面技术男。规划感很强的技术男，人不错，聊的很开心。
  • ls, 二面技术男。可能出身高，经验丰富。挺傲娇，聊完，让我等。行不行没有当面说，走了跟一面男说，然后一面男电话通知我。蒙蒙的从他们乱七八糟的小路里摸索出楼。这次，直接把面试心情拉到低点，感觉不会再爱了：）回家看到闺女的一刻，才抛开不爽。

补充半年的记录空白

Posted on 2016-02-25 Edited on 2021-08-13 In other 
    生活给予我们的是一种经历，掺杂的情感由自己控制。悲欢离合，怒喜思悲恐。所谓一粒沙里看世界，又何尝不是千人眼中的哈姆雷特？
    我，要更加努力，永不止步。

    开发环境从win、linux、个人笔记本、公司电脑不断切换，日记环境也不统一。落下不少，心理有愧，今儿是办完离职的第一天，填上这个文字债。继续前行...

从2015年年底到今天的叨咕

  1. 一个做了两年的计划，终于完成了。结果虽然不是完美，至少是一步。事情不是很容易，感谢家人的支持，是你们让我在最暗淡的时光，有信念、有坚持、有欢乐、有成长。
  2. 从12月份开始着手离职的事情。8年，也是我自己最好的8年时光，如果说得失参半算是充胖子，勉为其难的用用。事情都有两面性，从医学转到互联网行业，从0开始学习，点滴在心。我想如果要算得益，学习能力、知识宽度、思维（年龄因素更多？可能）都算对得起IQ。差的方面也很多，光说感悟吧：做完好事儿要留名；质量和结果要兼顾。8年，接触了来来往往不少同事，靠口才，靠耍宝，靠行业本领，靠学历，各种吃饭才能的都见过，也学到很多。
  3. 谈谈8年以来第一次接触面试。人，真正有本领的人，非常的谦逊。聊过几个国内第一学府的人，也聊过几个BAT出来的。傲娇不可取，待人勿如斯；公司，更多的跟你聊饼，其实很简单，咱们先谈要求和基本待遇。融资、上市这些，介绍一下即可，如果跟个人发展有直接关联，也可简介。 见过绝大部分公司是，一年赚几百、几千万，跟打工者半毛关系没有。最近看黄秋生的访谈，说，拍艺术片可以，但是别跟我光谈艺术，没钱一切都是假的。没钱，没有时间体验思考艺术。
  4. 说下北京和天津行业环境的差别，现在真正感受到差距。行业细分带来更多的是专业和深度，也是帝都高薪价值所在。

暂如此，想到再补充。

IPhelper简类

Posted on 2015-08-18 Edited on 2021-08-13 In tech

基础助手类，设计包含配置加载/ip数字互转/ip归属地查询/访问者ip获取/ip的段归属验证。

        <?php 

        class IPhelper {

            private static $banlist = array();

            private static $banedlist = array();

            static function loadConfig($banlist=array()) {
            self::$banlist = $banlist;
            }

            static function ip2num($ip) {
            if(strpos($ip,'.')===false) return 0;
            return bindec( decbin( ip2long($ip)));
            }

            static function getip() {
            if (getenv('HTTP_CLIENT_IP')) {
                $ip = getenv('HTTP_CLIENT_IP');
            } elseif (getenv('HTTP_X_FORWARDED_FOR')) {
                $ip = getenv('HTTP_X_FORWARDED_FOR');
            } elseif (getenv('HTTP_X_FORWARDED')) {
                $ip = getenv('HTTP_X_FORWARDED');
            } elseif (getenv('HTTP_FORWARDED_FOR')) {
                $ip = getenv('HTTP_FORWARDED_FOR');

            } elseif (getenv('HTTP_FORWARDED')) {
                $ip = getenv('HTTP_FORWARDED');
            } else {
                $ip = $_SERVER['REMOTE_ADDR'];
            }
            return $ip;
            }
            static function isBannedByiP($ip) {
            if(empty(self::$banlist)) return false;
            foreach(self::$banlist as $item) {
                $_item = explode('.',$item);
                $_sip = $_eip = null;
                if(count($_item)==3) { // xx.xx.xx
                $_sip = self::ip2num($item.'.0');
                $_eip = self::ip2num($item.'.255');
                }
                if(strpos($item,'-')) { // xx.xx.xx.xx-yy.yy.yy.yy
                $_item = explode('-', $item);
                $_sip = self::ip2num($_item[0]);
                $_eip = self::ip2num($_item[1]);
                }
                if($_sip && $_eip) {
                $_ip = self::ip2num($ip);
                if ($_ip>=$_sip && $_ip<=$_eip) {
                    return true;
                    break;
                }
                }
                if(count($_item)==4 && $item==$ip) { // xx.xx.xx.xx
                return true;
                break;
                }
                $_item = null;
            }
            return false;
            }

            static function isBannedClient () {
            $_ip = self::getip();
            if(isset(self::$banedlist[$_ip])) return true;
            $_isban = self::isBannedByiP($_ip);
            if($_isban) self::$banedlist[$_ip] = 1;
            return $_isban;
            }
        }


        // example
        $banlist = array(
            '192.168.0.1',  // 单ip
            '192.168.1',    // 网段
            '127.0.0.2',    // 单ip
            '127.0.0.0-127.0.0.255' // 网段
        );

        // 初始化加载配置
        IPhelper::loadConfig($banlist);

        $isban = IPhelper::isBannedClient();

        if($isban) {
            echo 'IP:' . IPhelper::getip() . ' is baned';
        }
        ?>
